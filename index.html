Do you want to be an IT student?

Yes? Then this event will showcase to you the innovation, fun, and teamwork to complete challanges and gain experience of the possibilities of a career in IT.

IT Code Fair is the perfect place for IT students to show off their skills to others or to learn more from others to always be improving on their skills.

WHERE? IT code fair

DoubleTree By Hilton Hotel Esplanade Darwin (116 Esplanade, Darwin City)

WHY? IT CODE FAIR

Amazing challenges and competitions to take part in. Showcase your coding skills to fellow students, teachers and judges. Solve real-life local business issues, win attractive prizes and more. Learn more about the competitions and events taking place.

What is IT Code Fair?

CDU IT Code Fair is a yearly event hosted by Charles Darwin University, where students are able to share their IT skills of all experiences, to interact and create digital problem solving programs to help others while connecting students with to each other, and idustry professions in the career field of IT.
